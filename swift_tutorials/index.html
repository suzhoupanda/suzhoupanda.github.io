Build an API Client that Uses Linked Lists to Download JSON data Build an API Client that Connects to the Quandl REST API endpoint
In this tutorial series, we build an API client that connects to the REST API endpoints developed and maintained by Last10k.com developers. This API provides SEC filing data for different companies, including balance sheets, income statements, and statements of cash flows. Optional query parameters allow you to get annual 10-K and quarterly 10-Q statements. We will build an API client that uses linked lists to make a series of API requests, whose returned JSON-data can then be used to create and save NSManagedObjects in CoreData.
