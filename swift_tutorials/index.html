In this tutorial series, we develop a 3D endless runner in SceneKit. We define a linked list generator class with customizable paramteres for developing linked lists, which in turn can provide the data needed to generate a series of enemy encounters during gameplay.
In this tutorial series, we build an API client that connects to the Oxford Dictionary REST API. Learn more about the URLSession class, which provides asynchronous methods for processing URLRequests. We also define a delegate protocol whose methods can be used by our API client, unit testing class, as well as a JSON-parser class that can be designed to process the JSON data returned from the server.
